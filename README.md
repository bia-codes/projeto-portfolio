🚀 **Projeto Portfólio**

 Este projeto é o resultado prático da meta de Aprendizado Aplicado e Compartilhado do Q1, focado na aplicação dos conceitos aprendidos no curso HTML e CSS para Iniciantes da Origamid.  
 


🛠️ **Tecnologias e Ferramentas**

 Linguagem: HTML semântico (com foco em acessibilidade) e CSS



🧠 **Evolução, Aprendizados e Desafios Superados**

 Antes deste projeto, eu não sabia exatamente como organizar pastas e arquivos de forma clara, como estruturar bem layouts usando grid e flex e utilizava diversas cores e fontes "soltas" no código, sem uma organização prévia. Com o estudo aplicado no curso, aprendi a organizar melhor as pastas e arquivos, a estruturar melhor todo o código e a armazenar certos valores em variáveis para torná-los reutilizáveis, o que melhorou significativamente a qualidade dos meus códigos e diminuiu a quantidade de linhas utilizadas para escrever comandos simples, deixando assim o código mais conciso e compreensível.



🏗️ **Estrutura do Projeto**

    assets/

🚀 **Como Executar o Projeto**

1. Criar a pasta do projeto
   
   Escolha um local no seu computadorclique com o botão direito e crie uma nova pasta (ex: meu-portfolio).

3. Abrir o terminal dentro da pasta
   
   Windows: Entre na pasta, clique na barra de endereços no topo, digite cmd e dê Enter.
   
   Mac/Linux: Clique com o botão direito na pasta e selecione "Abrir no Terminal".

3. Clonar o repositório
   
   Copie e cole esse comando no terminal:
   
       git clone https://github.com/bia-codes/projeto-portfolio.git

5. Visualizar o código
   
   Abra a pasta com o VS Code para visualizar o código.
